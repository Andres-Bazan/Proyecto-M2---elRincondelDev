Expert Python

Autor: Caso Parra, Astor

Editorial: Anaya Multimedia

Idioma: Español

Fecha de publicacion: 12/02/20

Sinopsis:

Python se utiliza en una amplia gama de dominios debido a su naturaleza simple pero poderosa. Aunque escribir código Python es fácil, hacerlo legible, reutilizable y fácil de mantener puede ser un desafío. Completa con las mejores prácticas, herramientas útiles y estándares implementados por desarrolladores profesionales de Python, esta cuarta edición lo ayudará no solo a superar tales desafíos, sino también a aprender las últimas funciones y conceptos avanzados de Python. El libro comienza con una preparación, donde se pondrá al día con las últimas mejoras de Python, elementos de sintaxis y herramientas interesantes para aumentar la eficiencia de su desarrollo. Además, los pocos capítulos iniciales deberían permitir a los programadores experimentados que provienen de diferentes lenguajes aterrizar de manera segura en el ecosistema de Python.

Web Design Playground

Autor: Caso Parra, Astor

Editorial: Anaya Multimedia

Idioma: Español

Fecha de publicacion: 02/04/21

Sinopsis:

Web Design Playground lo lleva paso a paso desde escribir su primera línea de HTML hasta crear páginas web interesantes y atractivas. En este libro basado en proyectos, utilizará un espacio de trabajo en línea personalizado, el área de juegos complementaria del libro, para diseñar sitios web, páginas de productos, galerías de fotos y más. La compra del libro impreso incluye un libro electrónico gratuito en formatos PDF, Kindle y ePub de Manning Publications. ¿Cómo aprenden los mejores diseñadores a crear hermosas páginas web y experiencias de usuario intuitivas? ¡Grandes ejemplos, tutoría experta y mucha práctica! Escrito por el diseñador web y maestro Paul McFedries, este libro único da forma y agudiza sus habilidades en HTML, CSS y diseño de páginas web.

Foundations HTML5

Autor: Caso Parra, Astor

Editorial: Anaya Multimedia

Idioma: Español

Fecha de publicacion: 12/09/19

Sinopsis:

Fundamentos de diseño y desarrollo web con HTML5 presenta temas de HTML y CSS como configuración de texto, configuración de color y diseño de página, con un enfoque mejorado en los temas de diseño, accesibilidad y estándares web. El texto relaciona tanto las habilidades necesarias (como HTML5, CSS y JavaScript) como las habilidades blandas (diseño, comercio electrónico y estrategias de promoción) consideradas fundamentales para el desarrollo web contemporáneo. El énfasis en la práctica práctica proporciona una base sólida para ayudar a los lectores en su carrera como profesionales de la web. La novena edición actualizada y ampliada contiene cobertura actual de HTML5 y HTML5.1, cobertura ampliada de diseño para dispositivos móviles y más.
